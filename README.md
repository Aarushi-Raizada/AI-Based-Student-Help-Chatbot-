# AI-Based Student Help Chatbot (CLI)

## 📌 About the Project

This is a simple chatbot made using Python. It is designed to help students with common problems like exams, stress, time management, CGPA, etc.

The chatbot works in a command-line interface (CLI) and gives responses based on keywords entered by the user.

---

## 🎯 Why this Project

As a student, I have seen that many students face similar problems again and again. Instead of searching every time, I thought of making a small chatbot which can give quick suggestions.

---

## 🧠 Concept Used

This project is based on rule-based AI.
It does not use machine learning. It simply checks user input and matches it with predefined keywords.

---

## ⚙️ Features

* Simple CLI-based chatbot
* Gives helpful and detailed responses
* Stores chat history in a JSON file
* Menu-driven system (easy to use)
* Help option to see topics

---

## 🛠️ Requirements

* Python 3

---

## ▶️ How to Run

1. Download or clone this repository
2. Open terminal in the folder
3. Run the file:

python SourceCode.py
