# Project Statement

## Project Title
AI-Based Student Help Chatbot (CLI Application)

---

## Problem Description
In college life, students face many common problems like stress, exams, time management, and confusion in studies. Getting quick and simple guidance is not always easy.

---

## Solution
To solve this, I developed a simple chatbot using Python. The chatbot can answer basic student-related queries using predefined rules. It works in a command-line interface and gives quick responses.

---

## Approach Used
The chatbot is based on a rule-based system. It checks user input and looks for keywords like "exam", "stress", "time", etc. Based on the keyword, it gives a suitable response.

---

## Key Features
- Easy to use CLI interface
- Provides guidance on common student problems
- Stores chat history using JSON file
- Menu-driven system

---

## What I Learned
While doing this project, I understood how basic AI works. I learned how rule-based systems function and how chatbots give responses based on input.

---

## Challenges Faced
One challenge was deciding the correct keywords for different queries. Sometimes the chatbot was not giving proper response if the input was different. I also worked on saving chat history correctly.

---

## Conclusion
This project helped me understand how a simple chatbot can be built using basic AI concepts. It is useful for students and easy to implement.
